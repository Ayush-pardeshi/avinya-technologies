1. Clone the repository:

2. Navigate to project:

## Git Workflow (SSH & GPG)

### Initial Setup
```bash
# Generate SSH key
ssh-keygen -t rsa -b 4096 -C "your.email@example.com"

# Generate GPG key
gpg --full-generate-key

# Configure Git
git config --global user.name "Ayush Pardeshi"
git config --global user.email "your.email@example.com"
git config --global user.signingkey KEYID
git config --global commit.gpgsign true
```

### Daily Workflow
```bash
# Create feature branch
git checkout -b feature/your-feature

# Make changes with VI editor
vi filename.html

# Stage and commit with GPG signing
git add .
git commit -m "Description of changes"

# Push via SSH
git push origin feature/your-feature
```

## Deployment

### GitHub Pages

1. Create repository: `avinya-technologies`
2. Push to `main` branch
3. Go to Settings → Pages
4. Select "Deploy from a branch"
5. Choose `main` branch and `/root` folder
6. Site will be live at: `https://Ayush-pardeshi.github.io/avinya-technologies`
